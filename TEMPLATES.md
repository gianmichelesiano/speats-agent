# Template Agenti Speats

Ogni template definisce skills, SOUL.md e cron predefiniti per un settore.

## 🏥 Template Farmacia

**Target:** Farmacie svizzere
**Costo:** 50-100 CHF/mese

### Skills
- `pharmacy-ai-assistant` — DB farmaci svizzeri (Swissmedic + OpenFDA)
- Turni e pianificazione
- Scadenze e report giornalieri
- Triage (29 algoritmi)
- VaxCheck (controllo vaccinazioni)

### Cron
- Report scadenze giornaliero
- Check aggiornamenti normativi

### SOUL
Professionale, preciso, in italiano/tedesco a seconda del cantone.

## 🍝 Template Ristorante

**Target:** Ristoranti, bar, take-away
**Costo:** 30-50 CHF/mese

### Skills
- Monitoraggio recensioni (Google, TripAdvisor)
- Gestione menu e orari
- Social media posting
- Prenotazioni (via Telegram)

### Cron
- Check recensioni ogni mattina
- Social post programmati

### SOUL
Amichevole, accogliente, tono caloroso.

## 🛠️ Template Consulenza

**Target:** Studi, freelance, PMI
**Costo:** 80-150 CHF/mese

### Skills
- `code-review` — review codice (da mattpocock)
- `github-issues` — gestione ticket
- `research` — ricerca tecnica
- `to-spec` / `to-tickets` — specifiche
- `handoff` — handoff tra sessioni

### Cron
- Backup monitoring
- Check aggiornamenti di sicurezza

### SOUL
Tecnico, diretto, problem-solver.
